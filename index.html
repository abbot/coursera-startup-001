CloudBoiler is a new smart and geeky way to manage your home heating system. Instead of using dumb timers, put yourself in control of your boiler.

CloudBoiler provides a highly tunable set of heating profiles, which can be adjusted in real-time by a distributed set of thermometers in your house.

CloudBoiler also provides a restful service API reachable through the internet (which can be used to control your appliances remotely)

CloudBoiler is compatible with most gas central heating systems. It can be connected in parallel to your existing temperature and timer switches, so you can even use your old thermostat as a prat of central heating control system.

CloudBoiler is built on top of Arduino and SparkCore platforms, and also can use ZigBee low-power wireless for connecting to external temperature sensors.

All software components of CloudBoiler rig will be released under an open source license.
